# Learning / EdTech Design Movements

Use for learning apps, edtech platforms, puzzle apps, flashcards, course products, interactive lessons, and educational games.

## Movement Entries

### Zen Minimalist Grid

- Definition: Puzzle/learning UI that removes decoration and centers numbers, letters, or grid logic.
- Visual cues: Off-white or soft gray backgrounds, thin borders, readable sans type, hidden secondary controls.
- Best fit: Sudoku, word puzzles, logic grids, calm daily brain-training.
- Use carefully / avoid: Avoid if the product relies on reward spectacle or character motivation.
- Request UI rationale hints: Choose when focus and low fatigue matter; tradeoff is low personality.

### Craft & Tactile Paper

- Definition: Paper-and-pencil-inspired UI that makes digital solving feel analog and tactile.
- Visual cues: Paper grain, graphite marks, sketch effects, page-turn transitions, soft natural ink.
- Best fit: Logic puzzles, journaling learning, handwriting-adjacent practice, analog-style study.
- Use carefully / avoid: Avoid overusing texture where contrast or crisp grids matter.
- Request UI rationale hints: Choose when analog comfort supports concentration; tradeoff is texture/asset care.

### Cozy Low-Poly / Watercolor Garden

- Definition: Reward-based learning UI where progress grows a gentle garden, town, or cozy scene.
- Visual cues: Watercolor gradients, low-poly plants/animals, soft pastels, calm ambient reward screens.
- Best fit: Healing puzzles, nonograms, casual learning streaks, low-pressure practice.
- Use carefully / avoid: Avoid when learning outcomes need strict professional credibility.
- Request UI rationale hints: Choose when motivation comes from calm accumulation; tradeoff is extra reward-world scope.

### Gamified Playful Flat

- Definition: Mainstream learning-game UI that makes study feel like clearing stages.
- Visual cues: Bright primary colors, mascot presence, roadmap stages, hearts/streaks, vector celebration.
- Best fit: Language learning, child education, daily drills, habit-based practice.
- Use carefully / avoid: Avoid for adult professional learning that needs restraint.
- Request UI rationale hints: Choose when retention and positive feedback matter; tradeoff is lower adult seriousness.

### Structured Scandinavian Minimalism

- Definition: Calm structured learning UI for adult education, technical courses, and high-text platforms.
- Visual cues: Warm white, olive/deep navy, progress bars, clean cards, multi-pane course layouts.
- Best fit: Courses, coding practice, certification, professional upskilling.
- Use carefully / avoid: Avoid for children or entertainment-led learning.
- Request UI rationale hints: Choose when concentration and trust matter under information load; tradeoff is less playfulness.

### Flashcard Tactile Spatial

- Definition: Study UI that treats cards as physical objects to flip, swipe, and sort.
- Visual cues: Centered cards, depth shadows, swipe left/right states, flip transforms, simple mastery colors.
- Best fit: Vocabulary, concept memorization, spaced repetition, quiz practice.
- Use carefully / avoid: Avoid when content is too dense for single-card units.
- Request UI rationale hints: Choose when memory decisions are binary and tactile; tradeoff is limited layout density.

### Interactive Sandbox Canvas

- Definition: Learning UI based on direct manipulation, pan/zoom, split views, and immediate simulation feedback.
- Visual cues: Infinite canvas, draggable blocks/shapes, sliders, toggles, real-time result panes.
- Best fit: Math, science, coding, design, systems thinking, interactive labs.
- Use carefully / avoid: Avoid novice overload by providing strong defaults and reset controls.
- Request UI rationale hints: Choose when learners must experiment, not just read; tradeoff is interaction complexity.

### Bouncy Vector Geometry

- Definition: Geometry-based playful education system using rounded primitive shapes and mascot-driven feedback.
- Visual cues: Rounded rectangles/circles/triangles, saturated friendly palette, no gray-heavy dead states, bouncy motion.
- Best fit: Child learning, language apps, early math, mascot-led practice.
- Use carefully / avoid: Avoid excessive motion and color conflict.
- Request UI rationale hints: Choose when the brand needs joyful consistency and low intimidation; tradeoff is a childlike tone.

### Atomic Scalable Informationalism

- Definition: Componentized education UI that scales curricula, quizzes, progress, and localization through atomic modules.
- Visual cues: Reusable badges, progress bars, lesson cards, quiz modules, consistent templates, translation-safe layouts.
- Best fit: MOOC platforms, large course catalogs, corporate learning, multi-language education.
- Use carefully / avoid: Avoid overly generic module grids that hide learning priority.
- Request UI rationale hints: Choose when the platform must scale many content types; tradeoff is reduced brand expressiveness.

### Immersive Scroll-Learning / Cinematic Education

- Definition: Scrollytelling education UI where scrolling drives layered animation, 3D scenes, or documentary-style explanation.
- Visual cues: Scroll-reactive infographics, 3D/video backgrounds, pinned text, staged reveals.
- Best fit: Public education, museums, science explainers, premium lessons, campaign learning pages.
- Use carefully / avoid: Motion/accessibility and performance risk; not ideal for repeat daily drills.
- Request UI rationale hints: Choose when one topic needs memorable immersion; tradeoff is engineering and content cost.

### High-Contrast Neo-Brutalist EdTech

- Definition: Raw high-impact education UI using bold borders, loud colors, and asymmetric grids to fight boredom.
- Visual cues: Thick black borders, neon accents, deliberate asymmetry, raw panels, assertive labels.
- Best fit: Coding bootcamps, design academies, youth-targeted learning, challenge campaigns.
- Use carefully / avoid: Avoid for regulated, clinical, or calm concentration-heavy learning.
- Request UI rationale hints: Choose when attention and attitude matter; tradeoff is fatigue and lower institutional polish.
