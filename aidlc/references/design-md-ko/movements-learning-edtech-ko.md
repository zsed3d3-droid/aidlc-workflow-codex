# Learning / EdTech 디자인 사조

Learning app, edtech platform, puzzle app, flashcard, course product, interactive lesson, educational game에 사용한다.

## Movement Entries

### Zen Minimalist Grid

- Definition: 장식을 제거하고 숫자, 글자, grid logic을 중심에 두는 puzzle/learning UI.
- Visual cues: off-white 또는 soft gray background, 얇은 border, 읽기 쉬운 sans type, 숨겨진 secondary control.
- Best fit: Sudoku, word puzzle, logic grid, calm daily brain-training.
- Use carefully / avoid: 제품이 reward spectacle 또는 character motivation에 의존하면 피한다.
- Request UI rationale hints: focus와 low fatigue가 중요할 때 선택한다. tradeoff는 낮은 personality.

### Craft & Tactile Paper

- Definition: digital solving을 analog/tactile하게 느끼게 하는 paper-and-pencil inspired UI.
- Visual cues: paper grain, graphite mark, sketch effect, page-turn transition, 부드러운 natural ink.
- Best fit: Logic puzzle, journaling learning, handwriting-adjacent practice, analog-style study.
- Use carefully / avoid: contrast나 crisp grid가 중요한 곳에서 texture를 과도하게 쓰지 않는다.
- Request UI rationale hints: analog comfort가 concentration을 돕는 경우 선택한다. tradeoff는 texture/asset 관리 부담.

### Cozy Low-Poly / Watercolor Garden

- Definition: progress가 부드러운 garden, town, cozy scene으로 자라는 reward-based learning UI.
- Visual cues: watercolor gradient, low-poly plants/animals, soft pastel, calm ambient reward screen.
- Best fit: Healing puzzle, nonogram, casual learning streak, low-pressure practice.
- Use carefully / avoid: learning outcome에 엄격한 professional credibility가 필요하면 피한다.
- Request UI rationale hints: calm accumulation이 motivation을 만드는 경우 선택한다. tradeoff는 reward-world scope 증가.

### Gamified Playful Flat

- Definition: study를 stage clear처럼 느끼게 하는 mainstream learning-game UI.
- Visual cues: 밝은 primary color, mascot presence, roadmap stage, heart/streak, vector celebration.
- Best fit: Language learning, child education, daily drill, habit-based practice.
- Use carefully / avoid: restraint가 필요한 adult professional learning에는 피한다.
- Request UI rationale hints: retention과 positive feedback이 중요할 때 선택한다. tradeoff는 adult seriousness 감소.

### Structured Scandinavian Minimalism

- Definition: adult education, technical course, high-text platform을 위한 차분하고 structured learning UI.
- Visual cues: warm white, olive/deep navy, progress bar, clean card, multi-pane course layout.
- Best fit: Course, coding practice, certification, professional upskilling.
- Use carefully / avoid: children 또는 entertainment-led learning에는 피한다.
- Request UI rationale hints: information load 아래에서 concentration과 trust가 중요할 때 선택한다. tradeoff는 낮은 playfulness.

### Flashcard Tactile Spatial

- Definition: card를 flip, swipe, sort하는 물리적 object처럼 다루는 study UI.
- Visual cues: centered card, depth shadow, swipe left/right state, flip transform, simple mastery color.
- Best fit: Vocabulary, concept memorization, spaced repetition, quiz practice.
- Use carefully / avoid: content가 single-card unit에 담기엔 너무 dense하면 피한다.
- Request UI rationale hints: memory decision이 binary이고 tactile할 때 선택한다. tradeoff는 제한된 layout density.

### Interactive Sandbox Canvas

- Definition: direct manipulation, pan/zoom, split view, immediate simulation feedback 기반 learning UI.
- Visual cues: infinite canvas, draggable blocks/shapes, sliders, toggles, real-time result panes.
- Best fit: Math, science, coding, design, systems thinking, interactive lab.
- Use carefully / avoid: 강한 default와 reset control을 제공해 novice overload를 피한다.
- Request UI rationale hints: learner가 읽기만 하는 것이 아니라 실험해야 할 때 선택한다. tradeoff는 interaction complexity.

### Bouncy Vector Geometry

- Definition: rounded primitive shape와 mascot-driven feedback을 쓰는 geometry-based playful education system.
- Visual cues: rounded rectangles/circles/triangles, saturated friendly palette, gray-heavy dead state 없음, bouncy motion.
- Best fit: Child learning, language app, early math, mascot-led practice.
- Use carefully / avoid: excessive motion과 color conflict를 피한다.
- Request UI rationale hints: joyful consistency와 낮은 intimidation이 필요할 때 선택한다. tradeoff는 childlike tone.

### Atomic Scalable Informationalism

- Definition: curricula, quiz, progress, localization을 atomic module로 확장하는 componentized education UI.
- Visual cues: reusable badge, progress bar, lesson card, quiz module, consistent template, translation-safe layout.
- Best fit: MOOC platform, large course catalog, corporate learning, multi-language education.
- Use carefully / avoid: learning priority를 숨기는 overly generic module grid를 피한다.
- Request UI rationale hints: platform이 많은 content type으로 확장되어야 할 때 선택한다. tradeoff는 brand expressiveness 감소.

### Immersive Scroll-Learning / Cinematic Education

- Definition: scroll이 layered animation, 3D scene, documentary-style explanation을 구동하는 scrollytelling education UI.
- Visual cues: scroll-reactive infographic, 3D/video background, pinned text, staged reveal.
- Best fit: Public education, museum, science explainer, premium lesson, campaign learning page.
- Use carefully / avoid: motion/accessibility/performance risk가 있고 repeat daily drill에는 적합하지 않다.
- Request UI rationale hints: 한 topic을 memorable하게 몰입시켜야 할 때 선택한다. tradeoff는 engineering/content cost.

### High-Contrast Neo-Brutalist EdTech

- Definition: bold border, loud color, asymmetric grid로 boredom을 깨는 raw high-impact education UI.
- Visual cues: thick black border, neon accent, deliberate asymmetry, raw panel, assertive label.
- Best fit: Coding bootcamp, design academy, youth-targeted learning, challenge campaign.
- Use carefully / avoid: regulated, clinical, calm concentration-heavy learning에는 피한다.
- Request UI rationale hints: attention과 attitude가 중요할 때 선택한다. tradeoff는 fatigue와 낮은 institutional polish.
